# ISKCON Event Ticketing System

A premium event booking platform with guest checkout and professional PDF ticket generation.

## 🚀 Features
- **Frictionless Guest Checkout:** No account needed, just enter Name, Email, and Phone.
- **Vintage Ticket Design:** Beautiful horizontal "Classic Stub" PDF tickets inspired by vintage kirtan events.
- **Automated Emails:** Instant booking confirmation with attached PDF tickets.
- **Ticket Lookup:** Guests can retrieve their past bookings using just their email.
- **Responsive Design:** Fully optimized for mobile and desktop.

## 🛠️ Technology Stack
- **Frontend:** React (Vite), Tailwind CSS, Framer Motion.
- **Backend:** Node.js, Express.
- **Database:** MongoDB Atlas.
- **Payments:** Razorpay (Test Mode active).
- **PDF Generation:** PDFKit.

## 📦 Installation
1. Clone the repo: `git clone https://github.com/santhosh12b/iskcon.git`
2. Install dependencies for both `client` and `server`:
   ```bash
   cd client && npm install
   cd ../server && npm install
   ```
3. Set up your `.env` file in the `server` folder.
4. Run the project:
   - Backend: `node server.js`
   - Frontend: `npm run dev`
